# Compilarea și rularea unui program C\#

Pentru a compila un program C#, aveți mai multe opțiuni:

* prin apăsarea de taste **CTRL+Shift+B**
* prin navigarea către meniul **Build** și selectarea opțiunii **Build Solution**

<figure><img src="/files/RtL8taEcTYpL7XdaBgj1" alt=""><figcaption></figcaption></figure>

Pentru a rula un program C#, aveți mai multe opțiuni:

* lansează programul cu debugger, ceea ce înseamnă că puteți folosi breakpoint-uri și executa fiecare linie de cod pas cu pas. Puteți face asta:
  * prin apăsarea tastei F5
  * prin navigarea către meniul **Debug** și selectarea opțiunii **Start Debugging**
  * prin apăsarea butonului **Start**, cu iconița arătând un triunghi verde plin
* lansează programul fără debugger, ceea ce este util atunci când vreți pur și simplu să vedeți rezultatul.&#x20;
  * prin apăsarea combinației de taste CTRL+F5
  * prin navigarea către meniul Debug și selectarea opțiunii Start Without Debugging
  * prin apăsarea butonului cu iconița arătând un triunghi verde gol
